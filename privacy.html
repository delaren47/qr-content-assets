Cualquier consulta sobre este aviso puede dirigirse a alfredo.del.arenal@gmail.com.

2. Propósito de la aplicación

Delaren Ads CLI es una herramienta interna de línea de comandos utilizada por Delaren Consulting para administrar campañas publicitarias de sus clientes en las plataformas de Meta (Facebook, Instagram, WhatsApp Business). La aplicación se conecta a la Marketing API y Pages API de Meta para automatizar tareas de gestión, optimización y reporteo de anuncios.

3. Datos que la aplicación accede

  • Datos agregados de desempeño de anuncios: impresiones, clics, alcance, conversiones de mensajería, gasto publicitario.
  • Metadatos de campañas, conjuntos de anuncios y anuncios: nombres, fechas, presupuestos, estado activo/pausado, identidades de página/cuenta.
  • Contenido orgánico de páginas de Facebook administradas por los clientes: publicaciones, fotos públicas, reacciones y comentarios agregados, únicamente con fines de análisis para optimización de creativos publicitarios.
  • Identificadores de cuentas de anuncios y negocios a las que el usuario operador tiene acceso autorizado dentro de Meta Business Manager.

La aplicación no accede ni almacena información personal identificable (PII) de usuarios finales que interactúan con los anuncios ni datos sensibles. Toda interacción con la API se realiza utilizando tokens de acceso autorizados por los propietarios legítimos de los activos en Meta.

4. Uso, almacenamiento y compartición de datos

  • Los datos consultados se utilizan exclusivamente para la gestión interna y reporteo de campañas a los clientes de Delaren Consulting.
  • No se realiza venta, alquiler, ni compartición de datos con terceros ajenos al cliente propietario de los activos publicitarios.
  • Los datos transitan localmente entre la API de Meta y la estación de trabajo del operador autorizado.
  • Cualquier respaldo o exportación se realiza únicamente para el cliente respectivo y se conserva en sistemas controlados por Delaren Consulting.

5. Derechos del titular de los datos

Los propietarios legítimos de las páginas, cuentas publicitarias o negocios cuya información sea consultada por la aplicación pueden en cualquier momento:

  • Revocar el acceso de la aplicación desde la configuración de su Meta Business Manager.
  • Solicitar la eliminación de cualquier respaldo local enviando un correo a alfredo.del.arenal@gmail.com.
  • Solicitar información sobre qué datos se han consultado o procesado en su nombre.

6. Eliminación de datos

Para solicitar la eliminación de cualquier dato relacionado con el uso de esta aplicación, envíe un correo electrónico a alfredo.del.arenal@gmail.com con el asunto Solicitud de eliminación de datos — Delaren Ads CLI. La eliminación se ejecutará en un plazo no mayor a 30 días naturales.
